# src/

Core registry implementation.

## Modules

| Module | Description |
|--------|-------------|
| `api/` | FastAPI REST & GraphQL server |
| `models/` | Pydantic data models for entities and rights |
| `validators/` | Rights-claim validation logic |
| `ledger/` | Append-only event log (immutable audit trail) |
| `cli/` | Command-line management interface |

See individual module docstrings for usage details.
